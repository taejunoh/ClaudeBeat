Features
Everything at a glance
Real-time
5-hour session, 7-day weekly, and extra usage credits with live reset countdowns
Display Modes
Session only, weekly only, or both in your menu bar with percentage and reset time
Notifications
Configurable threshold alerts for each metric, plus session reset notifications
Auto-refresh
Polls every 15 seconds to 5 minutes. Set it and forget it
Dashboard
Circular gauges, Sonnet breakdown, extra usage in dollars - all in a popover
Security
Keychain storage, app sandbox, Verified by Apple. Credentials never leave your machine

One command
$ brew install --cask taejunoh/tap/claudetokenusage
