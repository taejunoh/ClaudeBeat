Track Your Claude.ai Usage

Monitor your Claude AI token usage at a glance. A native macOS menu bar app for power users.

Download for macOS View on GitHub
Requires macOS 14.0 Sonoma or later · Apple Notarized

Everything you need to stay informed

Powerful features for Claude.ai power users who want complete visibility into their usage.

Real-time Monitoring

Track your 5-hour session limits, 7-day weekly usage, and extra usage credits in real-time with reset countdowns.

Three Display Modes

Show current session, weekly limit, or both in your menu bar. See usage percentage and reset time at a glance.

Smart Notifications

Configurable threshold alerts for session, weekly, and extra usage. Get notified when your session resets too.

Auto-refresh

Automatic usage updates every 15 seconds to 5 minutes. Set it and forget it - always up to date.

Detailed Dashboard

Click the menu bar to see circular gauges for session and weekly usage, plus Sonnet breakdown and extra usage in dollars.

Secure Storage

Session keys are stored securely in the macOS Keychain. Apple Notarized and sandboxed for your security.
